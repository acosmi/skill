# 百度 AI 搜索

使用百度 AI 搜索引擎（BDSE）进行网络搜索，获取实时信息、文档和研究内容，支持时间过滤和结果数量控制，专为中文内容优化。

## 概述

**百度 AI 搜索** 是一个调研分析领域的执行动作类技能。通过 4 个可配置参数实现灵活控制。

## 参数说明

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `query` | string | 是 | 搜索关键词 |
| `count` | integer | 否 | 返回结果数，默认 10 |
| `time_range` | string | 否 | 时间过滤 可选值: `day`, `week`, `month`, `year` |
| `site` | string | 否 | 站内搜索限定域名 |

## 输出格式

返回 JSON 对象，包含以下字段：

| 字段 | 类型 | 说明 |
|------|------|------|
| `success` | boolean | 操作是否成功 |
| `data` | object | 返回数据 |
| `message` | string | 结果描述 |

## 使用示例


```json
{
  "query": "示例搜索关键词",
  "count": null,
  "time_range": "day",
  "site": "示例站内搜索限定域名"
}
```

## 适用场景

- 需要搜索和整合多源信息时
- 学术研究或市场调研时
- 竞品分析或行业报告编写时

## 注意事项

- 技能标识: `baidu_search`
- 分类: 执行动作类 / 调研分析
- 必填参数: `query`
- 请确保输入参数符合 JSON Schema 规范
- 超时或异常时请检查参数格式和网络连接
