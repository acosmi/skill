# 股票基本面分析

使用结构化评分手册进行股票基本面分析与同行排名(质量、资产负债表安全、现金流等)。

## 何时使用

- 需要对单只或多只股票做基本面分析
- 需要按结构化手册打分与评级
- 需要做同行对比与排名
- 需要评估财务质量与稳健性

## 工作流

1. 开始前先阅读 `references/playbook.md`。
2. 严格按手册步骤执行:输入解析 → 数据收集 → 快速筛查 → 评分 → 评级 → 输出。
3. 多标的请求逐一处理并汇总排名。

## 常见陷阱

- 必须先读 playbook 再开始分析,不可跳步。
- 多标的需逐个完整评分,避免遗漏。
- 分析结论不构成投资建议。

## 输出

- 返回各标的的基本面评分、评级与同行排名。
