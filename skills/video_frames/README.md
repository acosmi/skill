# 视频帧提取

视频帧提取 — 使用 ffmpeg 从视频中提取指定帧或短片段，支持精确时间戳定位，适用于视频分析、缩略图生成和内容提取。

## 概述

**视频帧提取** 是一个多媒体处理领域的数据变换类技能。通过 6 个可配置参数实现灵活控制。

## 参数说明

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `input` | string | 是 | 视频文件路径 |
| `timestamp` | string | 是 | 时间戳（如 00:01:30）或秒数 |
| `duration` | number | 否 | 提取片段时长（秒） |
| `output` | string | 否 | 输出路径 |
| `fps` | number | 否 | 提取帧率，默认 1 帧/秒 |
| `format` | string | 否 | 输出格式 可选值: `png`, `jpg`, `gif` |

## 输出格式

返回 JSON 对象，包含以下字段：

| 字段 | 类型 | 说明 |
|------|------|------|
| `success` | boolean | 操作是否成功 |
| `data` | object | 返回数据 |
| `message` | string | 结果描述 |

## 使用示例


```json
{
  "input": "示例视频文件路径",
  "timestamp": "示例时间戳（如 00:01:30）",
  "duration": null,
  "output": "示例输出路径",
  "fps": null,
  "format": "png"
}
```
