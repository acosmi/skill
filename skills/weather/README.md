# 天气查询

天气查询 — 获取当前天气状况和多日预报，无需任何 API 密钥，开箱即用，支持全球城市查询。

## 概述

**天气查询** 是一个效率工具领域的执行动作类技能。通过 3 个可配置参数实现灵活控制。

## 参数说明

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `city` | string | 是 | 城市名称，支持中英文（如 北京、Tokyo、New York） |
| `days` | integer | 否 | 预报天数（1-7），默认 3 |
| `units` | string | 否 | 温度单位：metric=摄氏度, imperial=华氏度，默认 metric 可选值: `metric`, `imperial` |

## 输出格式

返回 JSON 对象，包含以下字段：

| 字段 | 类型 | 说明 |
|------|------|------|
| `city` | string | - |
| `current` | object | - |
| `forecast` | array | - |

## 使用示例


```json
{
  "city": "示例城市名称，支持中英文（如 北京",
  "days": null,
  "units": "metric"
}
```
