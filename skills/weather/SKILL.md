# 天气查询

获取实时天气与多日预报，开箱即用、无需 API 密钥。

## 何时使用

- 询问某地当前天气或气温
- 询问未来几天天气预报
- 出行/穿衣/带伞决策参考

## 工作流

1. 接收 `city`（城市名，必填）。
2. 可选 `days` 指定预报天数。
3. 可选 `units`：metric（摄氏）/ imperial（华氏）。
4. 返回当前状况与逐日预报。

## 常见陷阱

- city 用可识别的城市名；同名城市建议带国家/地区。
- days 过大可能超出数据源可提供范围。

## 输出

- 当前天气（温度、状况）与逐日预报列表。
