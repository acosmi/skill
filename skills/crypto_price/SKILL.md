# 加密货币价格

获取加密货币代币价格并生成蜡烛图(K 线图)。

## 何时使用

- 想查询某个代币当前价格
- 需要查看价格走势
- 想生成 K 线蜡烛图
- 快速了解币价行情

## 能力

- run:获取价格并生成图表
- query:查询指定代币价格
- configure:配置参数
- status:查看状态

## 工作流

1. 选择 action 并在 input 中提供代币名称或符号
2. 获取价格数据
3. 可生成蜡烛图
4. 按 output_format 返回结果

## 常见陷阱

- 代币符号拼写错误会查不到
- 行情数据有实时延迟
- 图表生成依赖足够的历史数据

## 输出

- 代币价格数据与 K 线蜡烛图
